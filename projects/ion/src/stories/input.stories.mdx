# Use em seu Projeto

Para usar IonInput em seu projeto, você também precisará importar os estilos predefinidos em sua aplicação Angular.

1. **Importe o estilo em seu arquivo SCSS:**

   ```scss
   @import '../node_modules/@brisanet/ion/styles/index.scss';
   ```

   Certifique-se de que o caminho para os estilos da biblioteca Ion esteja correto com base na estrutura do seu projeto. Sinta-se à vontade para ajustar os caminhos de importação.
